基本功能是让用户在校园BBS中发表留言、并查看留言；
用户注册与登录
实现用户注册注册时用户需要输入基本的个人信息;并以此注册信息登录论坛
查看留言列表
即留言列表模块，可分页显示用户留言标题，在该模块可仅列出留言标题及留言时间，并在标题上以超链接的形式，链接到具体页面，实现用户阅读留言
发表留言
登录用户才可以发表留言，留言板的设计需要包括留言主题和内容及时间等信息；
阅读留言
可通过链接实现某条具体留言信息的呈现，提取留言主题、内容及发表时间等
留言管理
实现管理员对留言的管理，主要是删除操作，对不合法的留言进行删除。

MySql数据库  M
用SQL脚本写完整的一个数据库系统，该脚本要保存
BBS数据库

用户信息表  Users
账号（char(12)，每一位0-9,not null,主键）    用户名    
密码  用户图像路径

管理员信息表    Admin   数据由人为插入
口令(char(12))

留言信息表  Message
留言号（int,自动生成,主键）  账号（char(12),外键）    
留言标题    留言时间（自动记录当前时间）    留言信息

回复信息表  Back
回复号（int,自动生成,主键）  留言号（int,外键）  
回复账号（char(12)，外键）     回复时间（自动记录当前时间）    
回复信息

存储过程
管理员登录验证功能Admin_Signin
    传入口令
    查询Admin表，看是否存在元组，满足传入的口令
        有，则放回1
        无，则返回0
用户登录验证功能User_Signin
    传入用户名密码
    查询Users表，看是否存在元组，满足传入的用户名密码
        有，则返回1，并返回账号
        无，则返回0
注册功能User_Signup
    传入用户名密码
    查询Users表，看是否存在元组，满足传入的用户名
        无，则将用户名，密码插入User表中，并放回1，并返回账号
        有，则返回0
个人图像功能User_img
    传入账号
    查询Users表，返回该账号的图像路径
主页留言查询功能Main_Query
    查询Message表和Users表的连接表，
    返回留言号，用户名，用户图像路径，留言标题，留言时间的结果集，
    按留言号降序排序
删除留言功能Main_Delete
    传入留言号
    删除Message表中该留言号的元组
    返回1
主页留言发表功能Main_Publish
    传入留言标题和留言信息和账号
    自动记录当前时间，自动生成留言号
    将留言号，账号，留言标题，留言时间，留言信息插入到Message表中
    返回1
用户图像修改功能User_imgUpdate
    传入账号和图像路径
    修改Users表中该账号的用户图像路径
    返回1
用户名称修改功能User_nameUpdate
    传入账号和用户名称
    修改Users表中该账号的用户名称
    返回1
用户密码修改功能User_passwordUpdate
    传入账号和用户密码
    修改Users表中该账号的用户密码
    返回1
个人页面功能User_Info
    传入账号
    查询Users表，返回该账号的用户名称，用户图像结果集
个人页面过去留言查询功能User_pastMessageQuery
    传入账号
    查询Users表和Message表的连接表，返回账号等于该账号的留言号，
    用户名，用户图像路径，留言标题，留言时间的结果集
个人页面过去回复查询功能User_pastBackQuery
    传入账号
    查询Back表和Users表和Message表的连接表，
    返回回复账号等于该账号的留言号，用户名，用户图像路径，
    留言标题，留言时间，回复号，回复信息，回复时间的结果集
个人页面过去回复删除功能User_pastBackDelete
    传入回复号
    删除Back表中该回复号的信息
    返回1
留言页面查询功能Message_Query
    传入留言号
    查询Messag表和Users表的连接表，
    返回该留言号的留言标题，留言时间，用户图像路径，用户名称，留言信息
留言页面回复查询功能Back_Query
    传入留言号
    查询Back表和Users表的连接表，返回该留言号的回复号，
    回复号的回复账号的回复用户名称，回复用户图像路径，
    回复时间（自动记录当前时间），回复信息

JavaServlet C
管理员登录功能Admin_Signin
    接收登录界面传来的口令信息
    调用数据库管理员登录验证功能存储过程Admin_Signin，传入口令
        若返回1，跳转到管理员主页
        若返回0，口令框下红色字体“口令错误”
用户登录功能User_Signin
    接收登录界面传来的用户名密码信息
    调用数据库用户登录验证功能存储过程User_Signin，传入用户名密码
        若返回1，为其创建一个Session对象，将账号密码保存到Session里面，跳转到主页
        若返回0，密码框下红色字体“用户名或密码错误”
注册功能User_Signup
    接收注册界面传来的用户名密码信息
    调用数据库注册功能存储过程User_Signup，传入用户名密码
        若返回1，刷新页面提示成功，为其创建一个session对象，将账号密码保存到Session里面，并跳转到主页
        若返回0，密码框下红色字体“该用户名已存在”
管理员主页留言删除功能Delete
个人页面过去留言删除功能
    接收传来的留言号
    调用删除留言功能存储过程Delete，传入留言号
        若返回1，弹出提示框“删除成功”
个人图像功能User_img
    接收Session对象账号
    调用嵌套页面个人图像功能存储过程User_img，传入账号
    接收到返回的图像并设置
主页留言查询功能Main_Query
    调用数据库主页留言查询功能存储过程Main_Query
    接收到返回的结果集
    按顺序添加
        留言栏
            div id = 留言号，
                a(div(p用户名 img用户图像) div(h1留言标题 p留言时间) href=留言的页面)
            判断Session中是管理员添加
            删除按钮id = 留言号
    分页显示
主页跳转与返回功能
    跳转时记录点击的留言号到Session对象中
    返回时从Session对象中清除该留言号
主页留言发表功能Main_Publish
    接收主页传来的留言标题和留言内容和账号
    调用主页留言发表功能存储过程Main_Publish，传入留言标题和留言内容和账号
        若返回1，弹出提示框“发表成功”
用户图像修改功能User_imgUpdate
    接收用户传来的图片信息和Session对象账号信息
    调用用户图像修改功能存储过程User_imgUpdate，传入图片信息和账号信息
        若返回1则，弹出提示框“修改成功”
用户名称修改功能User_nameUpdate
    接收用户传来的用户名称信息和Session对象账号信息
    调用用户名称修改功能存储过程User_nameUpdate，传入用户名称和账号信息
        若返回1则，弹出提示框“修改成功”
用户密码修改功能User_passwordUpdate
    接收用户传来的用户密码和Session对象的账号信息
    调用调用用户密码修改功能存储过程User_passwordUpdate，传入用户密码和账号信息
        若返回1则，弹出提示框“修改成功”
个人页面功能User_Info
    接收用户的Session对象账号信息
    调用个人页面功能存储过程User_Info，传入账号
    接收用户名称，用户图像路径结果集并设置
    将Session对象中的账号设置给个人页面
个人页面过去留言查询功能User_pastMessageQuery
    接收用户的Session对象账号信息
    调用个人页面过去留言查询功能存储过程User_pastMessageQuery，传入账号
    接收到返回的结果集
    按顺序添加
        留言栏
            div id = 留言号，
                a(div(p用户名 img用户图像) div(h1留言标题 p留言时间)href=留言的页面)
        删除按钮id = 留言号                
    分页显示
个人页面过去回复查询功能User_pastBackQuery
    接收用户的Session对象账号信息
    调用个人页面过去回复查询功能存储过程User_pastBackQuery，传入账号
    接收到返回的结果集
    按顺序添加
        留言栏
            div id = 回复号，
                a(div(p用户名 img用户图像) div(h1留言标题 p留言时间) div(p回复信息 p回复时间) href=留言的页面)
        删除按钮id = 回复号
    分页显示

个人页面过去回复删除功能User_pastBackDelete
